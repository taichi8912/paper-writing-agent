Read it to understand why the project exists and how it works.

- [design-philosophy.pdf](design-philosophy.pdf) (view in any browser)
- [design-philosophy.pptx](design-philosophy.pptx) (native, editable in PowerPoint)

1. Title

3. Objective and quality hierarchy
4. The eight design pillars
5. Technical core I: the anti-AI-slop linter
6. Technical core II: statistical honesty and citation grounding
7. Architecture: the hybrid core and plugin
8. Configurable, not rigid
9. Get started

## How it was built

The deck is native (directly editable) PowerPoint. Each slide was authored as a
1280x720 SVG (sources in [svg_output/](svg_output/)) and converted to native
DrawingML shapes, so every text box and shape is editable. No raster slide
images are used.
